# Scent Mapping

A minimal single-page site to map **scents → color → words**, saved locally in your browser.

## Run

Open `index.html` directly, or (recommended) run a tiny local server:

```bash
cd /Users/owochel/Desktop/Art/ScentMapping
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Data

- Stored in **Local Storage** under the key `scentMapping.entries.v1`
- Export anytime via **Download JSON** / **Download CSV**
